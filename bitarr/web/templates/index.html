Quick Start

Welcome to Bitarr! Here's how to get started:

  1. Perform a Scan - Click the "New Scan" button to select a directory to scan.
  2. View Results - After the scan completes, view the results to see file integrity status.
  3. Schedule Regular Scans - Set up scheduled scans to monitor your files automatically.
  4. Check Storage Health - Monitor the health of your storage devices over time.

Bitarr helps you detect file corruption, unauthorized modifications, and missing files across your storage devices.
